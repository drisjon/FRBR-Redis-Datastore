:mod:`lib.common` and :mod:`tests.test_common` Documentation
============================================================

Overview
^^^^^^^^
The :mod:`lib.common` module provides the base class from which 
other python modules use to interface with the `frbr-redis-datastore`
and useful standalone functions for interacting with 
`RDF <http://www.w3.org/RDF/>`_ XML and n3 file objects and the 
`Redis <http://www.redis.io>`_ datastore.

The :mod:`tests.test_common` are the unit tests for 
:mod:`lib.common`.
