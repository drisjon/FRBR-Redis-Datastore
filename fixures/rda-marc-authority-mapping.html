While there are many instances where there is a one-to-one correspondence between RDA and MARC 21, there are cases where the level of granularity in the RDA element set and MARC 21 element set differs. Where a single RDA element is equivalent to more than one MARC 21 element, all applicable MARC 21 elements are provided.
