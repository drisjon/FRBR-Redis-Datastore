About FRBR-Redis Datastore The technologies used in the FRBR-Redis Datastore Project

Redis

The FRBR-Redis Datastore implements the Entity-Relationship bibliographic theory called FRBR (Functional Requirements for Bibliographic Records) with open-source NoSQL key-value technology called Redis

Web frameworks

The FRBR-Redis Datastores uses the following web frameworks and javascript libraries to implement the user-interface

  • Twitter's Bootstrap
  • Raphael SVG Javascript library
  • 3-D Javascript three.js
