Quick Redis Setup

(Assumes you are running under a Linux VM)

Open a terminal window and run the following commands 

  $ wget http://redis.googlecode.com/files/redis-2.4.5.tar.gz
  $ tar xzf redis-2.4.5.tar.gz
  $ cd redis-2.4.5
  $ make

Now run the redis server

  $ src/redis-server
