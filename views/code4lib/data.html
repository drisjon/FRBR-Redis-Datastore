## MARC21 Section

MARC21

MARC21 is the current data carrier for library bibliographic records.

  • Legacy and wide-spread
  • Inoperability is a core requirement of LOC Bibliographic Framework
  • Colorado College's legacy ILS uses MARC21 for print holdings and purchased electronic resources

## START MODS Section

MODS

The Metadata Object Description Schema or MODS is an XML element set with language-based tags and attributes

  • Semantically richer than Dublin Core
  • Simpler than MARCXML and MARC21; however, does not provide full round-trip compatibility with MARC formats
  • Used by Colorado College's Fedora Commons Digital Archives
