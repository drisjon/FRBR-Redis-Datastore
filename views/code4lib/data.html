## MARC21 Section

MARC21

MARC21 is the current data carrier for library bibliographic records.

  • Legacy and wide-spread
  • Inoperability is a core requirement of LOC Bibliographic Framework
  • Colorado College's ILS uses MARC21 for print holdings and electronic resources

## START MODS Section

MODS

The Metadata Object Description Schema or MODS is an XML element set with language-based tags and attributes

  • Semantically richer than Dublin Core
  • Simpler than MARCXML and MARC21; however, does not provide full round-trip compatibility with MARC formats
  • Used by Colorado College's Fedora Commons Digital Archives hosted by Colorado Alliance of Research Libraries

## START FRBR Section

FRBR

Functional Requirements for Bibliographic Records or FRBR is an entity-relationship conceptual model for associating retrival and access user tasks with varying levels of abstract bibliographic records.

  • Adopted as part of RDA Vocabulary
  • Multiple mappings including FRBRoo, VRA, MARC, and MODS already exist or are being developed
