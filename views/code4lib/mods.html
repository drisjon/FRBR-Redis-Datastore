MODS

The Metadata Object Description Schema or MODS is a Library of Congress sponsor bibliographic XML schema defining elements and attributes including a subset of MARC 21 features.

XML, XPATH, and Redis Key-Value

To represent the MODS's XML structure in the key-value data structure of Redis, we use a modified version of the XML query language, XPATH, as a hash key for the MODS record. While this may sound complicated, it is actually easier to see as this examples show:
