LXML Module

The lxml is an xml python library used for MODS, RDA, SKOS, and general XML processing needs in the FRBR-to-Redis datastore project.

PyMARC Module

Ed Summer's PyMARC module is used in interaction of the FRBR-Redis datastore with MARC21 files

Redis.py Module

Andy McCurdy's Redis.py Python module is used for most utility and general datstore interaction with the C Redis store.
