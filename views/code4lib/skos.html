SKOS and RDA

Simple Knowledge Organization System Namespace (SKOS) along with an FRBR-based bibliographic rule-set called Resource Description and Access (RDA) provide the encoding for classes and properties for the FRBR RDA and FRBRoo Python classes. SKOS is also being leveraged as a general-purpose mapping tool for associating MARC21 and MODS data structures to FRBR entities.
